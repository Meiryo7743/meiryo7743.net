概要

簡単に「お腹が空きました！」と SNS へ投稿できる Web アプリです。主要な SNS やチャットアプリ等で実装されている OGP (Open Graph Protocol) の機能を用いることにより，onakasuitter の URL を貼り付けると「おなかすいた」という巨大な手書き風文字で空腹感を効果的に伝えられます。

お腹が空いて仕方ない一刻を争う事態のときに，わざわざ onakasuitter の URL を入力ことは大変不便なものです。そのため，Web Intents に対応している主要な SNS のボタンを押すとすぐさま投稿できるようにユーザーの利便性を考慮してあります。

技術

  • OGP：「おなかすいた」というメッセージ画像の表示
  • Web Intents：各種 SNS への投稿ボタン
  • Cloudflare Pages：ホスティング
